# AdGuardFilters

## AdGuard Content Blocking Filters

Ce dépôt contient une sélection **personnelle** de listes de filtres destinées **exclusivement à AdGuard**.

Les filtres sont soigneusement choisis et organisés afin d’améliorer le blocage des publicités, des domaines malveillants et d’autres contenus indésirables, tout en restant compatibles avec les règles et le moteur de filtrage d’AdGuard.

Ces listes sont utilisées dans un cadre personnel, mais peuvent être librement utilisées pour optimiser l’efficacité et la fiabilité de votre configuration AdGuard.
